Angular

This section will assume that you have done angular tour-of-heroes and build basic applications with angular, if not please go to angular website and read the documemtation. If you want to go straight to tour-of-heroes candidate, click here.

The aim is to create angular application and connect it to our backend (Node + Express = API) using HttpClientModule. With this connection, we will be able to interact with our backend from frontend. Let's begin! To see the development of ->API

Getting started

In order to start you need to have your angular CLI installed.

// to check if installed - check the version $ ng version // to install latest version - if not installed yet $ npm install -g @angular/cli

Once angular is installed, you ready to create your application and move forward. Let's create our CRUD (frontend) application.

// ng new [name of your application] $ ng new crud-api-application

This will generate your application, thereafter you can start working and incoperate your API.

Building Our Project

To build this project we will start by running some commands on the terminal to set our enviroment and our components to display features that we will need, we build our services to communicate with our api and functions that will handle the data from our api.

IN YOUR TERMINAL

APP MODULE

For us to be able to communicate with our API we need to import HTTP CLIENT MODULE from angular/common, this will enable http communication from our frontend to the backend through the API endpoints.

// src/app/app.module.ts import {{' {'}} HttpClientModule } from '@angular/common/http

In the same file go and register the imported module for it to be available in your angular application
